# App Store スクリーンショット（ジャマイカの木）

App Store Connect にアップロードする、**マーケティング仕様のスクリーンショット**一式。
実際に動作するアプリ（iOS シミュレータ）から撮影した**本物のスクリーンショット**を、
端末フレーム＋見出しコピー＋ブランドの「式の木」モチーフで構成した、人気ゲームアプリと同等の訴求画像。

## 収録画像（6枚・ストーリー順）

| # | ファイル | 見出し | 見せる画面 |
|---|---|---|---|
| 1 | `01-hero` | 数字をつなげて、目標の数をつくる。 | モード選択（ロゴ・アプリ名・2モード） |
| 2 | `02-core` | 5つの数字を組み合わせて解く。 | 盤面（式の木を育てる解答途中） |
| 3 | `03-solved` | 解けた瞬間の、この気持ちよさ。 | 盤面（正解！オーバーレイ） |
| 4 | `04-challenge` | 時間との勝負。何問解ける？ | チャレンジモード（残り時間・スコア11,400） |
| 5 | `05-difficulty` | こどもから大人まで、3つの難易度。 | 難易度選択（かんたん/ふつう/むずかしい） |
| 6 | `06-ranking` | 全国のプレイヤーと、スコアで競う。 | ランキング（ベストスコア・みんなのスコア） |

画面内の文言・数字・木の色（＋=mint / −=coral / ×=gold / ÷=purple）は**すべて実アプリのそのまま**。

## サイズ（Apple 公式規定準拠）

[Screenshot specifications](https://developer.apple.com/help/app-store-connect/reference/app-information/screenshot-specifications) に準拠。

| ディスプレイ | 解像度(px, 縦) | 用途 | 出力先 |
|---|---|---|---|
| iPhone 6.9″ | **1290 × 2796** | 新規申請の必須サイズ | `screenshots/iphone-6.9/` |
| iPhone 6.5″ | **1284 × 2778** | フォールバック | `screenshots/iphone-6.5/` |

> 6.9″ を用意すれば他サイズは App Store が自動スケールするため、最低限 6.9″ があれば申請可能。
> 本アプリは iPhone 縦画面のみ（iOS リリース）なので iPad サイズは未生成。

## デザインの根拠（＝「素人感・AI感」を出さないための設計）

- **本物のスクショを使用**：生成・模写ではなく、実機シミュレータで実際にプレイして撮影した画面。
  盤面は実際に解いて「式の木」が育った状態・正解演出・高スコアのチャレンジ結果を撮影。
- **AdMob テストバナーを排除**：端末が下フレームへ抜けるレイアウト＋クリップで、画面下部の広告を隠す。
- **一貫したテンプレート**：全6枚で見出し位置・端末フレーム・余白グリッド・光源を統一 → シリーズとしての一体感。
- **1枚1メッセージ**：情報を詰め込まず、太い見出し＋実画面の組み合わせに絞る。
- **ブランドの世界観**：背景にロゴの「式の木（カラフルなノードを線でつなぐ）」モチーフを配置。

## スクリーンショットの撮影と再生成

素材の実スクショは `generator/shots/*.png`（iPhone 17 Pro Max ネイティブ = 1320×2868、6.9″ 相当）。
撮り直したい場合はシミュレータで該当画面を出し、`xcrun simctl io <UDID> screenshot` で撮って
`generator/shots/<name>.png` を差し替える。合成の再生成は:

```bash
cd ModernJamaica/appstore/generator
./render.sh          # build.mjs で HTML生成 → Chromeで6.9"PNG → sipsで6.5"PNG
```

依存: Node（HTML生成・スクショを base64 埋め込み）、Google Chrome（ヘッドレスで正確なピクセルサイズに描画）、sips（macOS標準・縮小）。
見出しコピー・端末フレーム・クリップ量（`showPx`）は `generator/build.mjs` を編集して調整する。

## fastlane で申請にアップロードする場合

`fastlane deliver` を使うなら、解像度で自動判別されるため 6.9″/6.5″ の PNG を
`fastlane/screenshots/ja/` に置けばよい（本ディレクトリからコピー）。ファイル名の連番順で並ぶ。
